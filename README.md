# Build
```
npm install -g cordova grunt;
npm install;

grunt;
```

# Develop
```
grunt watch
```

# Run in Browser
```
./node_modules/serve/bin/serve ./www
```

# Run Cordova IOS
```
cordova run ios
```

# Run Cordova Android
```
cordova run android
```
